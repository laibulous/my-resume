# HTML Resume

This is a **basic HTML resume** created for practice while learning HTML fundamentals.  
It is intentionally **CSS- and JavaScript-free** to focus on semantic structure and proper HTML formatting.

---

## Live Demo
 [View Resume](https://laibulous.github.io/my-resume/)

## Project Overview

- Structured using **semantic HTML elements**
- Includes sections like About, Education, Skills, and Contact
- Deployed as a static website using **GitHub Pages**
- Serves as a simple, functional online resume

---

## Tech Stack

- **HTML5**
- **Git & GitHub**
- **GitHub Pages** (for deployment)

## What I Practiced

- Using semantic tags effectively
- Deploying a static site on GitHub Pages
- Linking content sections for easy navigation
